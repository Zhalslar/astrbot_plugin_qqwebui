## 💡 介绍

精心打造的第三方 QQ 客户端（网页版）,支持在 Astrbot 面板上与操控 QQ 进行交互

## 📦 安装

在astrbot的插件市场搜索astrbot_plugin_qqwebui，点击安装即可  

## ⌨️ 使用说明

在Astrbot仪表盘上打开本插件的面板即可体验
